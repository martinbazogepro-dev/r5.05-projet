# API Flashcards

API REST pour gérer des utilisateurs, des collections et des flashcards avec authentification JWT.


## Installation

1. Cloner le projet :
git clone <url-du-repo>

2. Lancer le serveur :
npm run dev

Le serveur tourne sur : http://localhost:3000

## Documentation API

La documentation Swagger est disponible à l’adresse : http://localhost:3000/docs
